# packer-kvm

Create VM templates for usage with libvirt/KVM virtualization

## Pre-requisites

 * libvirt/KVM
 * Packer (in /opt/packer)

## Distributions

* Centos 7.6

## Build

```
$ packer build centos7.json
```
